Pregunta 1: Una dirección que define cómo responde el servidor a una solicitud.
Respuesta: Una ruta (endpoint)

Pregunta 2: GET
Respuesta: Método HTTP utilizado para recuperar datos del servidor

Pregunta 3: POST
Respuesta: Método HTTP utilizado para enviar datos al servidor

Pregunta 4: Interfaz que permite que sistemas o aplicaciones se comuniquen entre sí.
Respuesta: API (Application Programming Interface)

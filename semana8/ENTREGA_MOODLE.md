## 🖥️ INFORMACIÓN DEL SISTEMA

### Comandos Ejecutados
- `uname -a` - Información del sistema operativo
- `whoami` - Usuario actual del sistema

### Explicación

El comando `uname -a` muestra información completa del sistema operativo, como el kernel, la versión, la arquitectura del procesador y el equipo.

El comando `whoami` muestra el nombre del usuario que tiene la sesión iniciada en ese momento.

Esta información es útil para administrar un servidor porque permite:
- Verificar el sistema operativo instalado
- Identificar el usuario con el que se está trabajando
- Facilitar tareas de mantenimiento y configuración
- Solucionar problemas de compatibilidad

## 🔐 BUENAS PRÁCTICAS DE SEGURIDAD DIGITAL

1. **Proteger las contraseñas** utilizando claves seguras y complejas.
2. **No compartir credenciales** de acceso con otras personas.
3. **Mantener el sistema operativo y los programas** siempre actualizados.

---

## 🛡️ PRÁCTICAS DE SEGURIDAD PARA PLATAFORMAS COMUNITARIAS

1. **No exponer datos personales** de los usuarios en acceso público.
2. **Utilizar contraseñas seguras** y proteger las credenciales de acceso.
3. **Acceder mediante conexiones seguras (HTTPS)** para encriptar la información.
4. **Registrar y monitorear los accesos** al sistema para detectar actividades sospechosas.
5. **Limitar el acceso a información sensible** únicamente a usuarios autorizados.

## 💭 REFLEXIÓN: IMPORTANCIA DE LA SEGURIDAD DIGITAL

La seguridad digital es fundamental para proteger la información de las personas y de las comunidades. Una plataforma segura evita el robo de datos y el acceso no autorizado. También ayuda a mantener la confianza de los usuarios al garantizar la privacidad de su información. El uso de contraseñas seguras y la actualización constante del sistema reducen los riesgos de ataques. Además, la protección de las credenciales evita que terceros utilicen las cuentas de forma indebida. Implementar buenas prácticas fortalece el funcionamiento de las plataformas tecnológicas. La seguridad es una responsabilidad compartida entre administradores y usuarios. Por estas razones, la seguridad digital es esencial para el correcto funcionamiento de cualquier plataforma comunitaria.
